# Hello-node.ts (Team4)

### How To Run This Project (for development)

- Get project by git clone
```
git clone..
```

- Download dependencies

```
cd dp-cars-club
npm install
```

- Add environment
```
cp .env.example .env
```

- Build and Run
```
docker-compose up --build -d
```

- Fresh logs
```
docker logs -f node_app
```
